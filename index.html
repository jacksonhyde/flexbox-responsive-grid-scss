Grid Layout

This is an example of a mobile-first responsive grid system built using Flexbox and BEM class notation.
To improve adoption rates, it is designed to follow the Bootstrap style DOM structure as closely as possible.

The grid is made up of three distinct elements:

  • .grid (the top level grid container)
  • .grid__row (an element for each grid row)
  • .col (the individual grid column elements)

.grid

This element is the top level grid component. Similar to bootstrap, you should only use one of these per grid (eg. a .grid element should never be nested under another .grid element).

.grid__row

This element is the container for all your grid columns. Combined with .grid, it uses the Boostrap negative margin trick to frame the layout and provide grid gutters. It should only ever be a child of a .grid element or a .col element.

.col

This is the main column class - all your grid content should sit within a col. Each .col element also needs at least one other class to determine the width of the column. These can be combined to create different layouts at different responsive breakpoints:

  • .col--xs-*
  • .col--sm-*
  • .col--md-*
  • .col--lg-*
  • .col--xlg-*
The * should be replaced by the number of columns you want the element to span. The grid is configured to use 12 columns by default.

Responsive Layout

The grid is configured to use the same media query breakpoints as Bootstrap:

  • $xsmall - 0px and up
  • $small - 576px and up
  • $medium - 768px and up
  • $large - 992px and up
  • $x-large - 1200px and up

These are used with the min-width CSS property to provide a Mobile First approach to a responsive DOM.
